## VSCode Great Icons

A comprehensive icon pack with 200+ custom file icons that make your file explorer easier to navigate at a glance.

**Install:**
1. Open VS Code
2. Press `Ctrl+Shift+X` to open the Extensions panel
3. Search for `VSCode Great Icons`
4. Click **Install**

Or install via the command palette (`Ctrl+Shift+P`):
```
ext install emmanuelbeziat.vscode-great-icons
```

**Link:** [Marketplace](https://marketplace.visualstudio.com/items?itemName=emmanuelbeziat.vscode-great-icons)
