# Components

## <img src="images/21stdev.png" width="24" alt="21st.dev"> 21st.dev

A community-driven component marketplace where you can explore, copy, and remix thousands of high-quality React components. Browse categories like heroes, buttons, pricing sections, AI chat components, testimonials, and more — all published by designers and developers. Great for quickly dropping polished UI into your projects.

**Link:** [https://21st.dev](https://21st.dev)

## <img src="images/shadcn.svg" width="24" alt="shadcn/ui"> shadcn/ui

A collection of beautifully designed, accessible, and customizable components built on Radix UI and Tailwind CSS. Unlike a traditional component library, shadcn/ui gives you the actual source code — you copy components into your project and own them completely. No dependency to manage, full control to customize. The go-to choice for modern Next.js and React apps.

**Link:** [https://ui.shadcn.com](https://ui.shadcn.com)
